# Data Sources

Synthetic source-style data for fintech growth, operations, product, and RevOps analytics.

- `entities.csv`: 36 growth initiative records.
- `daily_metrics.csv`: 5,040 growth initiative-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
